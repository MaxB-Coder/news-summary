## Screenshots

## Installation and Setup

1. **Clone the repository**:

   ```bash
   git clone https://github.com/MaxB-Coder/news-summary.git
   cd news-summary
   ```

2. **Install dependencies**:

   ```bash
    npm install
   ```

3. **Run the Server**:

   ```bash
    npm run dev
   ```

Visit the local host server that is in your console to view the project!

### Prerequisites

- Git
- Node.js

### Technologies

- React (Vite)
